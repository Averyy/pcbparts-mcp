Connect your client

Project scope 
claude mcp add --transport http jlcmcp https://jlcmcp.dev/mcp
User scope (all projects) 
claude mcp add -s user --transport http jlcmcp https://jlcmcp.dev/mcp

Add via Settings UI 
1. Open Claude Desktop
2. Go to Settings ‚Üí Connectors
3. Click "Add custom connector"
4. Enter name: jlcmcp
5. Set URL: https://jlcmcp.dev/mcp
6. Click "Add"
Or edit config: ~/Library/Application Support/Claude/claude_desktop_config.json 
{
  "mcpServers": {
    "jlcmcp": {
      "type": "http",
      "url": "https://jlcmcp.dev/mcp"
    }
  }
}
